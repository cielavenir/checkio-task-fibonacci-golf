Числа Фибоначчи или последовательность Фибоначчи - это числовой ряд:
0, 1, 1, 2, 3, 5, 8, 13, 21...
Но это не единственная интересная числовая последовательность; Есть немало похожих рядов, где каждый элемент получается из предыдущих. Взглянем на некоторые из них.

Вам нужно будет реализовать описанные ниже последовательности:

  • fibonacci: 
    f(0)=0, f(1)=1, f(n)=f(n-1)+f(n-2)
  • tribonacci: 
    f(0)=0, f(1)=1, f(2)=1, f(n)=f(n-1)+f(n-2)+f(n-3)
  • lucas: 
    f(0)=2, f(1)=1, f(n)=f(n-1)+f(n-2)
  • jacobsthal: 
    f(0)=0, f(1)=1, f(n)=f(n-1)+2*f(n-2)
  • pell: 
    f(0)=0, f(1)=1, f(n)=2*f(n-1)+f(n-2)
  • perrin: 
    f(0)=3, f(1)=0, f(2)=2, f(n)=f(n-2)+f(n-3)
  • padovan: 
    f(0)=0, f(1)=1, f(2)=1, f(n)=f(n-2)+f(n-3)

Дано имя последовательности и число "n". Вам нужно найти n-ый элемент данного ряда.

Главная цель в этой задаче написать ваш код, чем короче, тем лучше. Количество очков будет зависеть от длины вашего кода в скомпилированном виде. Короче код - больше очков. Для работы локально на вашем компьютере, можете использовать the local checker чтобы определить финальную длину кода. Очки за данную задачу динамические и определяются исключительно длиной кода. 1000 байтов - это граница за которую дается 0 очков. За каждый байт меньше 1000, вы получаете +1 очко. Для примера - 750 байт = 250 очков.

Входные данные: Два аргумента. Имя последовательности, как строка (str) и число "n", как положительное целое число.

Выходные данные: n-ый элемент, как целое число (int).

Как это используется: В этой задаче принцип "Не повторяйся" ("Don’t Repeat Yourself" (DRY)) пригодится вам, как нигде. это принцип разработки программного обеспечения, нацеленный на снижение повторения информации различного рода, особенно в системах со множеством слоёв абстрагирования. Принцип DRY формулируется как: «Каждая часть знания должна иметь единственное, непротиворечивое и авторитетное представление в рамках системы». Нарушения принципа DRY называют WET — «Write Everything Twice» (рус. Пиши всё по два раза). Это игра английских слов «dry» (рус. сухой) и «wet» (рус. влажный). Альтернативный вариант WET - "We Enjoy Typing"

Предусловия: 10 ≤ n < 400
